# ExpenseTracker

A simple Python project to track expenses.

## 📌 Overview
This project is a beginner-friendly Python program that helps users track their daily expenses. It allows adding expenses with categories, viewing all expenses, and calculating total spending.

## 🎯 Features
- Add expenses with category
- Display all expenses
- Show total spending

## 🧠 Concepts Used
- Lists
- Loops
- Calculations

## ⚙️ Requirements
- Python 3.x installed
- No external libraries required (basic version)
